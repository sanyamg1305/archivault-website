02
Approval Tracking

Every decision logged. Nothing ever disputed.

The single most expensive problem in architecture and interior design is the disputed approval. Archivault eliminates it entirely - every approval is timestamped, attributed to a named user, and permanently on record.

Approve any material or design inline with a single click
Every approval records the user and exact timestamp
Revision requests require a written explanation from client
Resubmit revised items for client re-review in one click
Full approval history viewable on every item
Design versions auto-supersede prior pending versions
Full version history available via history sheet
Client revision feedback visible per version
"Who approved it, when, and any notes attached." - always answerable.

Material Management

Every material decision tracked from selection to approval.

Interior design projects involve hundreds of material decisions. Archivault tracks every one - with images, costs, vendor details, approval status, and full history. No more "final kya decide hua tha?"

Add material with name, category, brand, vendor and cost
Attach reference image at creation or any point later
Vendor field renders as clickable link when URL entered
Search by name, brand, category or vendor
Filter by status: Pending / Approved / Revision Requested / Rejected
Client can upload or change material images from portal
Export full material schedule as print-ready PDF
PDF organised by room - hand directly to client or contractor

08
Security & Audit Trail

100% of actions logged. Enterprise-grade under the hood.

Every action on the platform is attributed to a named user with an exact timestamp. Nothing is ever lost, forgotten, or disputed. Security is enforced at the database level - no configuration required.

Multi-tenant: all data org-scoped at database level via Postgres RLS
Admins see all projects; clients only see their assigned project
Every action writes to activity log with user and timestamp
Last 15 actions surfaced on every project overview
Design files in org-scoped Supabase storage buckets
All file access controlled by user role, enforced server-side
Clerk authentication with signed JWT in every request
Client email invitation flow with auto-project assignment
100% of actions logged. 0 WhatsApp approvals. 1 place for everything.
